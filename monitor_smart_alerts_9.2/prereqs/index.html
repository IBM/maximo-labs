Pre-Requisite Instructions

Here are the pre-requisites for this Maximo Smart Alerts hands-on lab.

Attention

This lab requires Maximo Monitor 9.2

All Exercises

All Exercises require that you have:

  1. A computer with a Chrome, Firefox or Safari browser and internet connectivity.

  2. User access to a Maximo Application Suite environment with permissions to:

    • Access Maximo Monitor
    • Create and configure device types
    • Create and manage alerts
    • Administrator access to Maximo Application Suite (for advanced configurations)

  3. An IBM ID. If you don't have an IBM ID you can get one here:

    • Click Login to MY IBM button
    • Click Create an IBM ID link

  4. Test your access to the Maximo Application Suite environment.

Environment Setup

Ensure your Maximo Monitor environment has:

  1. Active Device Types: At least one device type with streaming data
  2. Data Flow: Devices sending data regularly to Monitor
  3. KPI Functions: Access to create and configure KPI functions

Verification Steps

Before proceeding with the exercises:

  1. Log in to Maximo Application Suite
  2. Navigate to Maximo Monitor
  3. Verify you can access Setup > Device Types
  4. Confirm you can view existing device data
  5. Check that you have permissions to create alerts

Tip

If you encounter access issues, contact your Maximo administrator to verify your user permissions and entitlements.
