Welcome to the Maximo Monitor 9.2
CSV Ingestion by Managed Gateway

Info

This lab demonstrates the Managed Gateway fallback system in Maximo Monitor and shows how data collection continues during network outages or MQTT connection failures.

In this lab, you will learn how the Managed Gateway automatically switches between data publishing methods, stores data locally during communication failures, and recovers data transmission when connectivity is restored..

The exercises will cover:

  • Understanding the Managed Gateway fallback hierarchy
  • Simulating MQTT connection failures
  • COS connection failure
  • PV Fallback and Primary Publisher Recovery

Note

Expected time needed to run the complete lab: 1 hour
